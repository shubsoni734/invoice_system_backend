# Skills

This folder contains reusable skill guides for working on the Invoice Backend project.

## Available Skills

- See `../CLAUDE.md` for the main project instructions.

## How to Add Skills

Create a new `.md` file in this folder named after the skill, e.g.:
- `add-domain.md` — step-by-step guide for adding a new API domain
- `migrations.md` — guide for writing and running Goose migrations
- `sqlc.md` — guide for writing sqlc-compatible SQL queries
